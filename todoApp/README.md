# Simple Todo App

A lightweight JavaScript application that allows users to create, display, and delete tasks dynamically in the browser.

## Features
* **No Page Reloads**: Uses `preventDefault()` for seamless, smooth interactions.
* **Input Validation**: Includes a guard clause to prevent users from adding blank tasks.
* **Dynamic Deletion**: Generates an individual, functional delete button for every unique list item.
* **Auto-Clear**: Resets the input field instantly after a task is added.
